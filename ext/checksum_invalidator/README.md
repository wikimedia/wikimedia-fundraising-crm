# Checksum Invalidator
Provides Contact::invalidateChecksum, an API action that marks as invalid a contact checksum, allowing it to be one time use or for admins to manually invalidate checksums.

This is an [extension for CiviCRM](https://docs.civicrm.org/sysadmin/en/latest/customize/extensions/), licensed under [AGPL-3.0](LICENSE.txt).

## Getting Started

Simply install the extension and then you can use Contact::invalidateChecksum(), either in code or via API Explorer.
